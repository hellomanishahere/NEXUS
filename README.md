**Nexus Gaming Hub - Modern Web Game Portal**

A modern game catalog and discovery platform for game enthusiasts to discover, explore and personalise their gaming experience. Inspired by the sleek, modern design of AAA game sites, it showcases mastery of fundamental web development by delivering a polished aesthetic alongside crucial dynamic features, built entirely using HTML, CSS, and Vanilla JavaScript. A capstone project demonstrating mastery of fundamental frontend web development by creating a responsive, interactive portal for accessing a curated list of games.

mobile).)

* **Responsive Layout:** The design is fully adaptive, ensuring a perfect viewing experience on mobile, tablet and desktop screens.
* **Dynamic Game Filtering:** Users can instantly filter the displayed games by platform (PC, Console, Mobile) to quickly find what they're looking for.
* **Interactive Game Carousel:** A modern, scrolling carousel showcases featured and trending games with a smooth, engaging user experience.
* **Search Functionality:** An efficient search bar allows for quick lookup of titles by name.
* **Aesthetic Design:** The interface uses modern typography, rounded corners and a clean color palette (utilizing Tailwind CSS principles) to provide a premium feel.
* **Project Structure:** The entire application is contained within a single index.html file, demonstrating efficient deployment and dependency management.
